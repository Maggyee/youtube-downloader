# 🎬 4K 视频下载神器

一个现代化的 YouTube 4K 视频下载器，支持多种画质、字幕下载和实时进度显示。

## ✨ 功能特性

- 🎨 **现代化 UI**：使用 customtkinter 构建，支持深色/浅色主题自动切换
- 🎯 **多画质支持**：4K/8K、1080p、720p、纯音频
- 📝 **字幕下载**：支持中英文字幕自动下载
- ⚡ **实时进度**：显示下载速度、进度百分比和剩余时间
- 🔧 **智能合并**：自动合并视频和音频流为 MP4 格式
- 🧵 **多线程**：下载时 UI 不冻结，体验流畅

## 📦 安装依赖

在项目目录下打开命令行，运行：

```bash
pip install -r requirements.txt
```

或者手动安装：

```bash
pip install customtkinter yt-dlp
```

## 🚀 运行程序

```bash
python main.py
```

## 📝 使用说明

1. **粘贴链接**：在输入框中粘贴 YouTube 视频链接
2. **选择画质**：从下拉菜单选择所需画质（默认最高画质）
3. **字幕选项**：如需字幕，勾选"下载字幕"选项
4. **开始下载**：点击"开始下载"按钮
5. **查看进度**：在日志框中实时查看下载进度
6. **完成**：视频会保存到程序所在目录

## ⚙️ 技术栈

- **UI 框架**：customtkinter - 现代化的 tkinter 增强库
- **下载引擎**：yt-dlp - 功能强大的视频下载工具
- **视频处理**：FFmpeg - 音视频合并（已包含在项目中）
- **多线程**：Python threading - 防止 UI 冻结

## 📂 文件说明

- `main.py` - 主程序文件
- `ffmpeg.exe` - 视频音频合并工具（必需）
- `requirements.txt` - Python 依赖列表
- `README.md` - 本说明文档

## ⚠️ 注意事项

1. 确保 `ffmpeg.exe` 在程序同目录下
2. 下载的视频会保存到程序所在目录
3. 下载大文件时请耐心等待
4. 请遵守 YouTube 服务条款，仅下载有权下载的内容

## 🐛 常见问题

**Q: 下载失败怎么办？**  
A: 检查网络连接，确认链接有效，尝试使用代理

**Q: 4K 视频下载后只有音频？**  
A: 确保 ffmpeg.exe 在正确位置，程序会自动合并

**Q: 字幕没有下载？**  
A: 部分视频可能没有中英文字幕，程序会尽力下载可用字幕
